# Private review notes

This directory is intentionally reserved for local, untracked portfolio review notes. `.gitignore` ignores everything in `private/` except this notice.

Do not store secrets, credentials, tokens, personal contact details, street addresses, private agreements, third-party confidential material, identifying screenshots, or proprietary code here. Gitignore is a convenience, not a security boundary.

Publicly useful decisions belong in `docs/decision-log.md`; implementation findings belong in tracked project documentation or issues.
